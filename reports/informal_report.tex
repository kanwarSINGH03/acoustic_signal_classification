\section{Introduction}
This is an informal report for the acoustic signal classification project. The goal of this project is to classify acoustic signals using machine learning techniques into "Drummy" or "Tight". The dataset used for this project is divided into 2 phases: Phase 1 and Phase 2.

\section{Stage 1}
\subsection{Data Analysis}
\subsubsection{Phase 1}
Phase 1 consists of \textbf{3,309 recordings} (\textbf{2,212 drummy} sounds and \textbf{1,097 tight sounds}).
Recordings in this phase have a duration of \textbf{0.75 seconds} and \textbf{36000 samples per recording}. The recordings are sampled at a rate of \textbf{48 kHz}. The dataset is not balanced, with number of drummy sounds twice that of tight sounds.
\paragraph{Findings}
\begin{itemize}
    \item Did a quick PCA analysis (first standardizing the data) and plotted the first 2 principal components. The 3D scatter plot (PC1 vs PC2 vs Label) shows that drummy and tight sounds are not linearly separable, and that tight sounds form a cluster in the middle of drummy sounds, while drummy sounds are more spread out.
    \begin{figure}[h!]
        \centering
        \includegraphics[width=0.4\textwidth]{../assets/p1_pca_3d.png}
        \caption{2 Component PCA Analysis of Phase 1 Data}
        \label{fig:phase1_pca_2_comp}
    \end{figure}
    \item Waveform visualization shows that drummy sounds dampen relatively slowly and have a longer decay time, while tight sounds dampen quickly and have a shorter decay time. This could suggest that sounds oscillated more in the hollow structure of the roof that produce the drummy sounds.
    \begin{figure}[H]

\subsection{Time-Domain Features}
These features are computed on short frames of the raw waveform and capture simple temporal properties.

\subsubsection{Zero Crossing Rate (ZCR)}
Extracted ZCR using 128-sample frames and 32-sample hops. Classification results:
\begin{itemize}
\item \textbf{SVM (RBF, C=100)}: Test accuracy = 0.7799
\item \textbf{MLP (1-layer)}: Test accuracy = 0.7832
\item \textbf{MLP (2-layer)}: Test accuracy = 0.8058
\item \textbf{MLP (3-layer)}: Test accuracy = 0.8155
\end{itemize}

\subsubsection{RMS Energy}
Extracted frame-wise RMS energy with 36-sample frames and 36-sample hops. Classification results:
\begin{itemize}
\item \textbf{SVM (RBF, C=100)}: Test accuracy = 0.8285
\item \textbf{MLP (1-layer)}: Test accuracy = 0.7832
\item \textbf{MLP (2-layer)}: Test accuracy = 0.8123
\item \textbf{MLP (3-layer)}: Test accuracy = 0.8414
\end{itemize}

\subsubsection{Amplitude Envelope}
Computed the amplitude envelope on 128-sample frames with 16-sample hops. Classification results:
\begin{itemize}
\item \textbf{SVM (RBF, C=100)}: Test accuracy = 0.7411
\item \textbf{MLP (1-layer)}: Test accuracy = 0.7638
\item \textbf{MLP (2-layer)}: Test accuracy = 0.7961
\item \textbf{MLP (3-layer)}: Test accuracy = 0.8123
\end{itemize}

\subsection{Raw Waveform Convolutional Classifier}
Trained a residual convolutional network directly on the raw waveform (36000 samples length) with stacked ResidualBlock layers. The model achieved:
\paragraph{Architecture}
The classifier is implemented as follows:
\begin{itemize}
\item 	extbf{ResidualBlock}: Each block applies two 1D convolutions (kernel size = 20, padding = 6), with an intervening ReLU activation. The output is optionally downsampled via a max-pooling layer (pool size and stride). A skip connection either uses identity or a 1×1 convolution (with matching stride and optional pooling) to align channel dimensions. The block output is the elementwise ReLU of the sum of main and skip paths.
\item 	extbf{Convolution Module}: The network begins with a Conv1d layer mapping 1→2 channels (kernel size =10, padding=4) and ReLU. This is followed by two ResidualBlock layers:
\begin{itemize}
    \item ResidualBlock(2→5, poolsize=10, stride=2)
    \item ResidualBlock(5→8, poolsize=10, stride=2)

\end{itemize}
After the residual blocks, we flatten the feature map and use two fully connected layers: 704→128 with ReLU and dropout (0.5), then 128→2 output logits.
\end{itemize}
\begin{itemize}
\item Test accuracy = 0.9644
\item Precision/Recall/F1 for drummy (0): 0.96 / 0.99 / 0.98 (support = 221)
\item Precision/Recall/F1 for tight (1): 0.96 / 0.91 / 0.94 (support = 88)
\end{itemize}
